# 🧪 Automação de Testes E2E com Cypress - SauceDemo

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar e aprimorar habilidades em **Quality Assurance (QA)** através da automação de testes End-to-End (E2E) utilizando o framework Cypress.

A aplicação utilizada para os testes foi o ambiente de demonstração SauceDemo, uma plataforma criada especificamente para treinamento de automação de testes, contendo usuários e cenários simulados para validação de funcionalidades de e-commerce.

O foco deste projeto foi aplicar boas práticas de automação, organização de testes e validação de fluxos críticos de uma aplicação web.

---

## 🚀 Tecnologias Utilizadas

- Cypress
- JavaScript
- Node.js
- Mochawesome (relatórios de testes)
- Git & GitHub

## 🎯 Objetivos do Projeto

Durante o desenvolvimento desta automação, foram trabalhados conceitos fundamentais da área de QA, como:

- Automação de testes End-to-End (E2E)
- Validação de fluxos críticos de negócio
- Estruturação de cenários de teste
- Escrita de testes legíveis e organizados
- Utilização de seletores CSS
- Manipulação de elementos da interface
- Validação de mensagens e comportamentos da aplicação
- Simulação de ações reais do usuário
- Geração de relatórios de execução
- Boas práticas com Cypress
- Controle de versão com Git

## 🛒 Funcionalidades Testadas

Os cenários automatizados contemplam funcionalidades importantes do e-commerce, como:

- Login com usuário válido
- Login com usuário inválido

## 💳 Checkout

- Preenchimento de formulário
- Finalização de compra
- Validação da mensagem de sucesso

- Navegação entre páginas
- Validação de URLs
- Validação de elementos visíveis

## ▶️ Como Executar o Projeto

### Clone o repositório

```bash
git clone https://github.com/joaoeduardo2701/e-commerce-automation-cypress/
```

### Acesse a pasta do projeto

```bash
cd nome-do-projeto
```

### Instale as dependências

```bash
npm install
```

### Execute o Cypress em modo visual

```bash
npx cypress open
```

### Execute os testes em modo headless

```bash
npx cypress run
```
